# v0.1.0-alpha Release Notes

`v0.1.0-alpha` is the first tagged public development release of `DevQuizAPI`.

## Highlights

- Local development environment is easier to bootstrap
- Quiz import and package image support are working end to end
- Admin upload/import workflows are clearer and more reliable
- A public sample quiz package is included in the repository
- Integration tests now cover the core auth, quiz, and import paths

## Included In This Release

- automatic development database migration on startup
- seeded development admin account
- seeded fake sample quiz data for fresh development databases
- root landing page and Swagger routing cleanup
- improved ZIP upload/import handling
- uploaded file listing fix
- quiz image import and display support
- `/health` endpoint for quick environment verification
- GitHub Actions CI for build and test validation
- public README and sample package documentation

## Known Alpha Limitations

- deployment is still manual
- release publication is still manual
- production deployment guidance is documented but not yet automated

## Recommended Audience

This release is appropriate for:

- local development
- internal evaluation
- limited technical preview use

This release is not yet positioned as a stable public beta.
